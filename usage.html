Usage Guide

To start working with this repository you need to clone it onto your local machine:

$ git clone https://github.com/patricia-ternes/hypot-2023
$ cd hypot-2023

Next we install the hypot package into the environment using setup.py:

$ python setup.py install

Using the functions

If you want to create a personalised script, you can import the hypot modules as follows:

from hypot.calc import squared, addition, sqroot
